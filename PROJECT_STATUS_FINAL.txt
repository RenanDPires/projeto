┌─ QUESTÃO 4: Condutores Retangulares de Cobre ───────────────────────────────┐
│                                                                               │
│ Status: ✅ COMPLETO                                                          │
│ Partes: (a) Deduções analíticas para 3 variantes                            │
│         (b) Cálculo da densidade superficial de perdas [W/m²]                │
│                                                                               │
│ VARIANTE (a): Campo em Ambas as Superfícies                                 │
│   Condição: H_z(±b) = H₀ (simetria)                                         │
│   Solução: H_z(x) = H₀·cosh(x/δ)                                            │
│   Perdas: P_a = (H₀²/σδ)·tanh(b/δ)                                          │
│   Aplicação: Condutor imerso em campo uniforme                               │
│                                                                               │
│ VARIANTE (b): Campo em Uma Superfície (Semi-espaço)                         │
│   Condição: H_z(0) = H₀, H_z(∞) = 0                                         │
│   Solução: H_z(y) = H₀·exp(-y/δ)                                            │
│   Perdas: P_b = H₀²/(σδ)  [CASO BASE]                                       │
│   Aplicação: Condutor em placa laminar                                       │
│                                                                               │
│ VARIANTE (c): Espaço Finito (Sanduíche)                                     │
│   Condição: H_z(0) = H₀, H_z(b) = 0                                         │
│   Solução: H_z(x) = H₀·sinh((b-x)/δ)/sinh(b/δ)                              │
│   Perdas: P_c = (H₀²/σδ)·coth(b/δ)                                          │
│   Aplicação: Sanduíche com confinamento bidimensional                        │
│                                                                               │
│ Dados da Questão 4:                                                          │
│   H₀ = 6.0 A/m (campo magnético na superfície)                              │
│   σ = 5.8×10⁷ S/m (condutividade do cobre)                                  │
│   b = 2.5 cm (meia-largura)                                                  │
│   2b = 5.0 cm (largura total)                                                │
│   f = 60 Hz (frequência)                                                     │
│   μᵣ = 1.0 (não-magnético)                                                   │
│                                                                               │
│ RESULTADOS NUMÉRICOS:                                                        │

│   b/δ = 2.9303 (razão adimensional)                                          │
│   tanh(b/δ) = 0.994317                                                       │
│   coth(b/δ) = 1.005716                                                       │
│                                                                               │
│   VARIANTE (a): P_a = 7.2338×10⁻⁵ W/m² (relativo: 0.9943 × P_b)            │
│   VARIANTE (b): P_b = 7.2752×10⁻⁵ W/m² (base = 1.0000)                      │
│   VARIANTE (c): P_c = 7.3168×10⁻⁵ W/m² (relativo: 1.0057 × P_b)            │
│                                                                               │
│   J_max(a) = 1.5398×10⁶ A/m² (em x = ±b)                                    │
│   J_max(b) = 1.6486×10⁵ A/m² (em y = 0)                                     │
