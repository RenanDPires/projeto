Cloudflare Worker Shell

Eletromag Lab pronto para deploy estático via GitHub.

Este repositório continua com a aplicação principal em Streamlit/Python para uso local, e agora também inclui um ponto de entrada compatível com Cloudflare Workers para publicação de uma vitrine estática do projeto pelo fluxo GitHub → Cloudflare.

O que este deploy faz

  • Serve uma página inicial estática no edge.
  • Redireciona rotas inexistentes para o índice.
  • Permite publicar o repositório diretamente do GitHub.

O que não faz

  • Não executa o app Streamlit completo dentro do Worker.
  • Não roda o núcleo Python pesado de simulação no edge.
  • Não substitui o ambiente local para cálculo científico.

Como usar

Envie este repositório para o GitHub, configure os segredos do Cloudflare e execute o deploy via Wrangler. O conteúdo em cloudflare/public será publicado como entrypoint estático. O restante do projeto permanece disponível para execução local e evolução posterior para uma arquitetura web completa.

Se a meta for hospedar a interface interativa completa no Cloudflare, será necessário portar a camada de UI e cálculo para uma stack compatível com Workers. Este scaffold resolve a publicação rápida do repositório e evita quebrar o fluxo atual em Streamlit.
